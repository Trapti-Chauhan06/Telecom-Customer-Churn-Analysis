# 📊 Telecom Customer Churn Analysis 

## 📌 Project Overview

Customer churn is a major challenge for telecom companies because losing existing customers directly impacts revenue and business growth.

This project analyzes telecom customer data using Python, SQL, and Machine Learning to predict customer churn and identify the key factors influencing customer retention. SHAP (SHapley Additive Explanations) is used to explain model predictions and improve model interpretability.

## 🎯 Project Objectives

- Predict whether a telecom customer is likely to churn.
- Analyze customer behavior using SQL and Python.
- Compare multiple Machine Learning models.
- Identify the most influential features using SHAP.
- Provide business recommendations to reduce customer churn.

## 📈 Key Insights

- Month-to-month contract customers are more likely to churn.
- Customers with shorter tenure have a higher churn rate.
- Monthly charges significantly influence customer churn.
- Contract type is one of the strongest predictors of churn.
- SHAP identified the most influential features affecting churn predictions.

## 💡 Business Recommendations

- Encourage customers to choose long-term contracts.
- Offer personalized retention plans for high-risk customers.
- Provide loyalty rewards for long-term customers.
- Improve customer support and service quality.
- Use predictive analytics to identify customers likely to churn.
